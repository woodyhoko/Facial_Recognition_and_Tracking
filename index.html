// IDENTITY-AWARE TRACKING SYSTEM

Recognize once.
Track forever.

Running face recognition on every frame is prohibitive — and fails the moment a subject turns away. This system runs recognition only when needed and hands spatial continuity to the ECO tracker: identity-aware tracking at a fraction of the compute.

// COMPONENTS
Three engines, one box.
[ DETECT ]

Viola-Jones

Cascade of boosted Haar-feature classifiers over an integral image. >30fps on CPU; fires every N frames or on track failure.

[ RECOGNIZE ]

PCA → FLDA

Eigenfaces compress 65K→128 dims; Fisherfaces maximize between-class scatter — robust to illumination. Runs once per new face.

[ TRACK ]

ECO-HC

Factorized correlation filter + GMM sample model + conservative updates. Propagates the box every frame, carrying the identity label.
